# Sales Operations Portfolio

Interactive tool demos built for sales strategy, territory design, and market analysis.

## Tools

| Tool | Description |
|------|-------------|
| [Territory Intelligence Planner](tools/territory-intelligence-planner.html) | ZIP-level territory assignment with CRM export |
| [Market Opportunity Dashboard](tools/market-opportunity-dashboard.html) | TAM analysis with upsell/cross-sell scoring |
| [Whitespace Command Center](tools/whitespace-command-center.html) | OEM whitespace and share-of-wallet analysis |
| [Sales Performance Map](tools/sales-performance-map.html) | Geo-layered rep attainment and account risk view |

## Stack

Plain HTML/CSS/JavaScript — no build step, no dependencies beyond CDN-loaded libraries (Leaflet.js, Chart.js, Inter/Space Grotesk fonts). Hosted on GitHub Pages.
